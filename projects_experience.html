STATISTICAL ANALYSIS IN COMPUTATIONAL MODELING OPTIMIZATION

REINFORCEMENT LEARNING IN POMDP

data-driven approach to high-level robot control

  • By analyzing the data of the behavior of the robot, we can gain a deep understanding of the way the robot behaves in the real world and therefore make a sound decision in controlling the robot.

CLASSIFICATION OF POWER PLANTS: CNN VS. GENERAL REGRESSION MODEL

  • Two different machine learning models were developed to predict the cooling types of 350 Indian power plants. General regression analysis was used to predict the plants using a dataset of multiple numerical and categorical labels and CNN was used to do the same using images from Google Map images.

MACHINE LEARNING APPROACH TO KINODYNAMIC MOTION PLANNING

  • In order for robots to make quick decisions online, they must make appropriate decisions depending on which state they are in. In this article, a framework for optimal control motion planning using optimization-based control and defining reachable sets using weighted linear regression will be discussed.
